# **App Name**: GlobeFlow – Immigration Command Center

## Core Features:

- Immigration Command Center Dashboard: Provides a comprehensive overview of the immigration journey with progress tracking, status updates, and prioritized 'next best actions' for the user.
- Smart Document Engine with AI Guidance: Facilitates efficient document management including secure upload, status tracking, review feedback, and an AI-powered tool providing specific guidance on how and where to obtain each required document.
- Secure Asset Transfer Portal: Manages the process of transferring assets from a home bank to Indonesia, featuring live status tracking, estimated arrival times, and transparent breakdown of fees and foreign exchange rates.
- Dedicated Case Manager Communication: Enables direct and discreet communication with a dedicated Relocation Case Manager via an in-app chat interface, appointment scheduling, and clear visibility of response time commitments.
- Comprehensive Cost Dashboard: Offers a simulated breakdown of estimated total transfer costs, including exchange rate impacts and fees, to ensure financial transparency throughout the immigration process.
- Multi-Language Interface: Allows users to seamlessly toggle the entire application's user interface between English and German for personalized linguistic support.
- Admin Operations Console: A restricted interface for administrators to manage user document and asset transfer statuses, add review comments, and respond to user inquiries from case managers.

## Style Guidelines:

- Color scheme: Light scheme, designed for clarity and a professional feel, akin to private banking interfaces. The choice of hues is influenced by trust and discretion.
- Primary color: A sophisticated, muted blue (#476685). This color evokes a sense of trust and reliability, fitting for financial services, and serves as a foundational element for premium branding.
- Background color: A very light, almost white, cool grey (#F7F7F8). This provides ample whitespace and a clean canvas, contributing to a discreet and modern aesthetic.
- Accent color: A fresh, yet muted, cyan (#69CCCC). This analogous color provides a subtle but distinct highlight for interactive elements and calls to action, maintaining contrast without being overtly bright or flashy.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for conveying complex information with clarity and professionalism across all text types.
- Use a set of minimalistic, clean line icons that communicate functionality efficiently and visually enhance the premium, trustworthy aesthetic. Avoid overly elaborate or cartoonish designs.
- Prioritize generous whitespace and a card-based layout structure for optimal readability and a sense of calm. Content blocks are clearly separated by subtle dividers or shadow effects, ensuring a premium and uncluttered look on both desktop and mobile. 'Next steps' are prominently displayed.
- Implement subtle, functional micro-interactions and smooth transitions that guide the user and provide delightful feedback without being distracting. Animations should feel precise and swift, contributing to the app's modern and premium feel.
